| Поле | Тип | Назначение |

| `Country` | string | Страна |
| `Channel` | string | Канал: sales / performance / organic |

### Несовпадение признаков возврата

- `3 255` строк с `Quantity < 0`, но без префикса `C`
- `1` строка с префиксом `C`, но `Quantity > 0`
- Pipeline использует комбинированное правило: `Quantity < 0` ИЛИ `Invoice LIKE 'C%'`

### Текстовый шум (устранён нормализацией)

- `184 479` описаний с лишними пробелами по краям
- `48 058` с множественными пробелами внутри

- `1 041` StockCode с несколькими вариантами Description

`EIRE → Ireland`, `USA → United States`, `RSA → South Africa`, `Unspecified → Unknown`, `European Community → European Community / Other`
